Ответы к Macmillan Practice Tests (1-е изд.)

Ответы к Macmillan Practice Tests по аудированию и чтению к изданию первому (Клековкиной, 2006), не ко второму (Вербицкой, 2009). Ответы разных изданий отличаются — чуда не будет.

Пособия серии Macmillan Exam Skills for Russia и Macmillan Exam Skills for Russia: Speaking and Listening обеспечивают подготовку учащихся к сдаче Единого государственного экзамена (ЕГЭ) по английскому языку.
