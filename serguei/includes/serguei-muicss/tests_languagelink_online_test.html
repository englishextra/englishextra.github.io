Уровневый тест

«Уровневый тест» с ответами по материалам languagelink.ru.

Цель

«Для того, чтобы подобрать программу, максимально отвечающую Вашим целям, специалисты академического отдела Language Link разработали тест по английскому языку. Он рассчитан на определение уровня с точки зрения лексики и грамматики. Поскольку уровень складывается из оценки всех языковых компетенций, в продолжения онлайн-теста нужно будет пройти устное тестирование с преподавателем-носителем в ближайшем учебном центре. По результатам письменного и устного тестирования эксперты Language Link назначают подходящий курс.»
